# M06 — GitOps con ArgoCD

[← Página anterior](../M05-cicd/M05-02-pipeline-cd.md) · [Siguiente página →](M06-01-intro-gitops.md)

## Qué aprenderás

- Instalar ArgoCD y conectar el repositorio como fuente de verdad.
- Sincronización automática ante cambios en `values.yaml`.
- Promoción controlada entre entornos con Git.

## Teoría

**GitOps**: el estado deseado del clúster vive en Git; un operador (ArgoCD) reconcilia continuamente. Los cambios son auditable y reversibles.
